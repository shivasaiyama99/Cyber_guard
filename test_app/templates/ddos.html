💡 How It Works

Attack Types

  • HTTP Flood — Sends rapid GET/POST requests including SQLi payloads, brute force login attempts, and path scanning
  • SYN Flood — Rapid TCP connect/disconnect to exhaust connection limits
  • Slowloris — Opens many connections and sends partial HTTP headers to hold them open
  • Mixed — Combines all attack types simultaneously

What Gets Detected

  • Cyberguard monitors same_ip_requests_per_minute (100 req/min threshold)
  • UFW firewall blocks trigger ddos_blocks_per_ip_per_minute (30/min)
  • Fail2ban bans trigger fail2ban_bans_per_ip alerts
  • SQLi payloads trigger sqli_attempts_per_ip (CRITICAL)
  • Failed logins trigger failed_login_per_ip_per_minute (HIGH)
  • HIGH/CRITICAL alerts send email via SMTP
